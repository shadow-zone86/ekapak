# Breadcrumbs Component

Компонент хлебных крошек для навигации по разделам сайта.

## Использование

```tsx
import { Breadcrumbs } from '@/shared/ui/breadcrumbs';

// Простые хлебные крошки
<Breadcrumbs
  items={[
    { label: 'Главная', href: '/' },
    { label: 'Корзина', active: true },
  ]}
/>

// Многоуровневые хлебные крошки
<Breadcrumbs
  items={[
    { label: 'Главная', href: '/' },
    { label: 'Категория', href: '/category' },
    { label: 'Товар', active: true },
  ]}
/>
```

## Props

### BreadcrumbsProps

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `BreadcrumbItem[]` | - | Массив элементов хлебных крошек (обязательно) |
| `className` | `string` | - | Дополнительные CSS классы |

### BreadcrumbItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Текст элемента (обязательно) |
| `href` | `string` | - | URL для ссылки (если не указан и элемент не активен, будет отображен как текст) |
| `active` | `boolean` | `isLast` | Активен ли элемент (по умолчанию последний элемент считается активным) |

## Поведение

- **Активные элементы** отображаются с синим фоном (`bg-blue-600`) и белым текстом
- **Неактивные элементы с href** отображаются как ссылки с белым фоном и рамкой, при наведении появляется эффект hover
- **Разделители** (`/`) автоматически добавляются между элементами
- **Последний элемент** по умолчанию считается активным, если не указано иное

## Примеры

```tsx
// Страница корзины
<Breadcrumbs
  items={[
    { label: 'Главная', href: '/' },
    { label: 'Корзина', active: true },
  ]}
/>

// Страница товара
<Breadcrumbs
  items={[
    { label: 'Главная', href: '/' },
    { label: 'Категория', href: '/category' },
    { label: 'Товар', active: true },
  ]}
/>

// Все элементы как ссылки (нет активного)
<Breadcrumbs
  items={[
    { label: 'Главная', href: '/', active: false },
    { label: 'Раздел', href: '/section', active: false },
  ]}
/>
```

## Доступность

Компонент использует семантический HTML (`<nav>`) с атрибутом `aria-label="Хлебные крошки"` для улучшения доступности. Разделители помечены как `aria-hidden="true"`, чтобы скринридеры не озвучивали их.
