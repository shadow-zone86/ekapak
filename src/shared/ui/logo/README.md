# Logo

Компонент логотипа приложения EKAPAK.

## Описание

`Logo` - это компонент для отображения логотипа компании. Использует Next.js Image компонент для оптимизированной загрузки изображения.

## Использование

```tsx
import { Logo } from '@/shared/ui/logo';

export function Header() {
  return (
    <header>
      <Logo />
    </header>
  );
}
```

## Props

| Prop | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| `className` | `string` | `undefined` | Дополнительные CSS классы для кастомизации стилей |

## Примеры

### Базовое использование

```tsx
<Logo />
```

### С кастомными размерами

```tsx
<Logo className="w-32 h-32" />
```

### Маленький логотип

```tsx
<Logo className="w-16 h-16" />
```

### Большой логотип

```tsx
<Logo className="w-48 h-48" />
```

## Технические детали

- Использует Next.js `Image` компонент для оптимизации
- Изображение загружается из `/logo.svg`
- Поддерживает все стандартные HTML атрибуты через spread оператор
- Адаптивный размер через Tailwind CSS классы

## Связанные компоненты

- `NavIcon` - иконки навигации
- `Button` - кнопки с логотипом
